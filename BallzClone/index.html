Ballz Clone

So this is a clone of the game Ballz by Ketchapp. I am making this for a few reasons. First, Learning how to make a web game. Second, the original has several issues or things that could make the game better and this allows me to make those. Third, this version will be cross platform and will work on any device that has a web browser.
